SpringBoot banner

发表于 2021-05-29 更新于 2021-10-16 分类于 Java ， Spring Boot

Spring Boot 启动时，控制台输出的图案叫 _banner_，如下所示的 Spring 图案。

最简单方式改写 banner 输出的图案

  1. 在 src/main/resources 目录下新建 banner.txt
  2. 在 banner.txt 上加上图案，可以在这个网站-一个 ASCII 图案网站-上获得自己想要的 ASCII 图案

这样在启动项目时控制台就可以打印出自己的图案了。

也可以使用 jpg、png 甚至 gif，都命名为 banner。

或者通过设置banner.image.location属性来作为 banner 信息，这些图片会被转换为有艺术感的 ASCII，并且打印在文本的顶部。
